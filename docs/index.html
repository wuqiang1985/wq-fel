WQ Web Front-end Library

安装

npm install wq-fel

用法

举个栗子，我们想调用encodeHtml

  • 引入整体库
    import wq-fel from "wq-fel";
// call
wq-fel.string.encodeHtml('xx');
  • 引入某个整体模块
    import string from "wq-fel/string";
// call
string.encodeHtml('xx');
  • 引入某个具体某个方法
    import encodeHtml from "wq-fel/string/encodeHtml";
// call
encodeHtml('xx');
